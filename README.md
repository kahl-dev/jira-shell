# jira-shell

> Get data from your jira (Current only worklogs)

## Install

```
$ npm install --global jira-shell
```

_Requires [Node.js](https://nodejs.org) 4+._

## First Run

Configure the plugin with `jira -c`

## Usage

Run: `jira -h`
